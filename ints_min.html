Welcome to the alpha release of the SGD interactions viewer!

This viewer shows "slices" of the Yeast BioGRID physical interaction network in a manner designed to convey biological insight. You must have the ADOBE FLASH plugin for your browswer installed.

To get started, enter a gene name (Standard or Systematic) in the Enter Gene (new network): box under the Network Info tab.

ALL comments, suggestions and bug reports are appreciated, you can report them on the GitHub tracker, or send an email to sgd-helpdes@lists.stanford.edu

The viewer uses CytoscapeWeb and YeastMine, with the interaction network assigned in a way that conveys Gene function and bilogical process.

The network viewer will show up in the empty space to the right of the control panel. The control panel can be lengthened or shortened by pulling on the edges.

The SGD Interactions viewer uses the protein-protein interaction network from BioGRID to draw graphs. Interacting "genes" (usually proteins but in some cases RNA) are represented by "nodes" - the colored shapes that you see, labeled with their Standard Gene name. The interactions between genes are represented by lines or arrows for differnt types of experiments.

Some general controls for panning and zooming the viewer can be found in the lower right hand corner of the display window. There are mouse-over tooltips if you are confused by the icons there

Nodes can be pulled around by click-dragging them if they are selected. You can click and drag outside a node (in the viewer) to create a "box selection" of nodes. To select a single node, click on it in the viewer. Orange higlights are drawn around all selected nodes. right clicking (option-click on a mac) a node will bring up a context menu with options to

  • Delete a node from the current network
  • Expandthe network around this node
  • Goto the SGD locus page for these gene (opens a new window)
  • Select the first neigbors of the node

right-clicking anywhere in the viewer will allow you to try some alternative network layouts of the current network.

The Reset Network button in the Network Info panel will re-run the initial layout algorithm (and repaint the original colors. There are also some flash player options in the context menu, use them at your own peril.

This interface is used to control network creation and addition. Networks are currently composed of a gene and all it's physical interacting neigbors (from BioGrid).

"Edges" or interactions between the neighbors are also drawn, but each neighbor likely has additional neighbors.

Enter a yeast gene standard or systematic name (names will autocomplete) in the "new network" box to create a new network (destroying any existing network). Doing the same in "add to network box" will get an additional gene's network (neighbor or not).

Currently, perfomance of the alpha version of the SGD Network viewer will degrade once several 100s of nodes and edges are drawn.

Adding to the network is also available from the gene (node) context (right-click) menu.

The Reset Network button will restore the original "party" colors and force-directed layout./

When a gene (node) is clicked in the viewer a mini SGD locus page appears here. Click on one of the three tabs to see General, GO, or Phenotype information.

The SGD Interaction Viewer (alpha) colors nodes based on their Yeast GO Slim Biological Process(es). Because most genes belong to more than one Biological Process, there is no unique way to color them. When the network is initially generated, colors vis-a-vis biological processes are assigned in an essentially arbitrary but reproducable manner.

The node coloring interface will let you "paint" a specific biological process with a color of your choosing. It is often informative to first color ALL nodes a neutral shade prior to painting.

The black numbers in brackets next to each Process indicate the number of nodes in the current network that will be painted

The "Reset Network" button in the Network Info tab will reset the current network to the original "rainbow" color scheme

Mousing over a node (hold for a brief moment) will show you the GO SLIM terms assigned to that node. The full GO annotations are available from Gene Info window when a node is clicked.

Nodes with a black border are "Black listed" because they have many hundreds of physical interactions that may not be specific or informative.

Node shapes are assigned by the Molecular Function of the gene they represent, according to the priority shown, top to bottom (i.e,. Protein Kinases are both Kinases and Signal tranducers and Enzymes, but they are assigned the Octogon shape.

Each edge ("interaction") represents one or more experiments linking two genes (nodes). There are currently 4 types of experiments displayed, with purple representing the majority of physical interactions in Biogrid. "Self" or "dimer" interactions are ignored except for Biochemical Activity. More BioGRID interaction types...

The thickness of each edge is proportional the the number of unique experiments supporting this interaction. Exact numbers can be found by mousing over the edge (or looking at the relevant gene in SGD
